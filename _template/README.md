# Project Template

This template folder is for creating new project pages.

## Usage

1. **Copy this folder** and rename it with a serial number (e.g., `202302`, `202303`)
2. **Replace images**:
   - `thumb.jpg` - Thumbnail for gallery (200px tall)
   - `hero.jpg` - Main project image
3. **Edit index.html**:
   - Update project title
   - Update year
   - Update external link URL and aria-label (or comment out if not needed)
   - Update icon if needed (currently points to `../thangs-logo.svg`)
   - Update description paragraph (or comment out if not needed)
4. **Update main gallery**: Add link to `/index.html` gallery section

## What to Edit

Look for comments in `index.html`:
- `<!-- EDIT: ... -->` - Required edits
- `<!-- OPTIONAL: ... -->` - Can be commented out or deleted if not applicable
